## List

A vertically stacked set of interactive headings that each reveal a section of content.

<div class='component-preview '>
  <List client:load />
</div>

```js
import { List } from '@iftakhar/ui';
const data = [
  {
    title: 'item 1',
    content: 'content',
  },
  {
    title: 'item 2',
    content: 'content',
  },
];
```

```js
<List
  data={data}
  renderItem={(item) => (
    <div>
      <h1>{item.title}</h1>
      <p>{item.content}</p>
    </div>
  )}
/>
```
